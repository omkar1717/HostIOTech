Body Posture Detection

Introduction: This project detects whether a person is sitting or standing using an accelerometer-based wearable device placed on the thigh. The system records time spent in each posture for health and ergonomics analysis.

Features:

  • Real-time posture detection
  • Data logging for sitting and standing durations
  • Compact and wearable device
  • Can be integrated with health monitoring systems
  • Potential applications in fitness and workplace ergonomics

Working Principle: The accelerometer measures the tilt of the thigh. Based on predefined threshold values, the system determines if the user is sitting or standing and logs the time spent in each position.
