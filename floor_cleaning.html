ML-Powered Floor Cleaning Robot

Introduction: A smart cleaning robot that uses Machine Learning & Image Processing to detect dirty areas and navigate autonomously. The robot identifies dirt or stains on the floor using a camera sensor, processes the images using OpenCV, and moves toward the detected dirt patches for cleaning.

Features:

  • AI-based dirt detection
  • Automated navigation and obstacle avoidance
  • Works on different floor surfaces
  • Smart cleaning path optimization
  • IoT integration for remote control

Working Principle: The robot scans the floor using image processing, detects dirty areas, and moves toward them for cleaning. ML algorithms improve efficiency over time.
